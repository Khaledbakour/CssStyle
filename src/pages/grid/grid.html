grid Layout

Einleitung

Grid ist ein CSS Modul, welches ein zweidimensionales Layout bereitstellt. Es ist besonders gut geeignet zur Strukturierung einer Website in verschiedene Bereiche und zur Gruppierung von Elementen. Grid sollte nicht mit der Flexbox verwechselt werden, welches in erster Linie ein eindimensionales Layout bietet. Flexbox kann zwar bei einem Überlauf an Elementen weitere Rows liefern, wenn man allerdings dynamische Eigenschaften der Flexbox deaktiviert, um z.B. Spalten mit anderen Elementen zu zentrieren, sollte zu einem Grid gegriffen werden.

Container und Elements

Ein Grid besteht aus dem Grid Container, welcher mehrere Child Elemente beinhaltet. Ein HTML Element wird zu einem Grid Container, indem man ihm per CSS die Eigenschaft display: grid oder display: inline-grid gibt.

Hier wird display: grid verwendet, um ein grid zu erstellen mit 4 Elementen. Dabei wird grid-template-columns 1fr 1fr 1fr verwendet, um die Anzahl und Breite der Spalten zu bestimmen. In diesem Fall gibt es 2 Spalten mit jeweils 1/3 Platz und eine Spalte mit 1/6 Platz des zu füllenden Elements.

Den Unterschied zwischen dem inline-grid und dem grid erkennt man an den Platzhaltern, welche Siblings zu den grids in deren Parent Element darstellen. Ein inline-grid gliedert sich zwischen den Siblings ein, während ein grid diese oben und unten positioniert.

Positionierung der Elemente

Es gibt verschiedene Wege Elemente innerhalb des Grids zu positionieren und sie über mehrere Spalten und Reihen ragen zu lassen.

Die Linien des Grids sind nummeriert und mithilfe von grid-column und grid-row lassen sich Elemente im Grid psoitionieren und über mehrere Spalten und Reihen strecken.
