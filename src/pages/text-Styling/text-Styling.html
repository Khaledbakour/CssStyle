text-styling

Einleitung

CSS bietet eine breite Palette von Eigenschaften zur Formatierung von Text, mit denen Sie das Aussehen des Texts auf verschiedene Weise steuern können. Diese Eigenschaften können verwendet werden, um die Schriftart, Größe, Farbe, Gewichtung, Ausrichtung und vieles mehr zu ändern

Color & Background color

Mit der Color-Eigenschaft können Sie die Farbe und die Hintergrundfarbe der Schrift ändern.

color

Hier haben wir zum Beispiel die Farbe rot verwendet, um die Schriftfarbe in rot zu ändern

p { color: red; }
Hello world!
Background-color: white

Hier haben wir die Hintergrundfarbe weiß verwendet, um die Hintergrundfarbe der Schrift zu ändern.

p { Background-color: white; }
Hello world!
RGB-Modell

Im RGB-Farbschema kann jede Farbe durch die genaue Kombination der Primärfarben Rot (R), Grün (G) und Blau (B) definiert werden. Auf diese Weise ermöglicht Ihnen das RGB-Modell, rund 16,7 Millionen unterschiedliche Farben für die Gestaltung Ihrer Website zu definieren. Hexadezimale Farbcodes

Das Grundschema eines hexadezimalen Farbcodes besteht aus einem Hashtag (#) gefolgt von sechs Hexadezimalzahlen, die in Zweierblöcken je eine Farbe des RGB-Farbraums (Rot, Grün, Blau) repräsentieren: #FFFFFF

font-family

Die font-family-Eigenschaft in CSS spezifiziert eine priorisierte Liste von einer oder mehreren Schriftfamilien und/oder generischen Familiennamen für das ausgewählte Element.

Web-Safe Fonts

Web-Safe Fonts sind Schriftarten, die auf den meisten Betriebssystemen vorinstalliert sind. Sie sind so gestaltet, dass sie auf jedem Gerät und in jedem Browser korrekt angezeigt werden. Durch die Verwendung dieser Schriftarten stellen Webdesigner und Entwickler sicher, dass die beabsichtigte Schriftart immer korrekt auf einer Webseite angezeigt wird, selbst wenn diese Schriftarten nicht auf dem Computer des Benutzers installiert sind.



Die folgenden Schriftarten sind die besten Web-Safe Fonts für HTML und CSS


  • Arial
  • Verdana
  • Times New Roman
  • Georgia
  • Courier New
p { font-family: Courier New, monospace; }
Hello world!
Default Fonts

In CSS gibt es fünf generische Schriftfamilien:


  • Serifenschriften haben einen kleinen Strich an den Rändern jedes Buchstabens. Sie vermitteln ein Gefühl von Formalität und Eleganz.
  • Serifenlose Schriftarten haben klare Linien (ohne kleine Striche). Sie vermitteln ein modernes und minimalistisches Aussehen.
  • Monospace-Schriften - hier haben alle Buchstaben die gleiche feste Breite. Sie erzeugen ein mechanisches Aussehen.
  • Kursive Schriftarten imitieren die menschliche Handschrift.
  • Fantasy-Schriften sind dekorative/verspielte Schriftarten.

font-size

Die font-size-Eigenschaft in CSS bestimmt die Größe der Schrift. Es gibt verschiedene Methoden, um die Schriftgröße in CSS festzulegen:

Pixel

Die Festlegung der Textgröße mit Pixeln gibt Ihnen die volle Kontrolle über die Textgröße. Zum Beispiel:


Die folgenden Schriftarten sind die besten Web-Safe Fonts für HTML und CSS

Absolut Beschreibung Beispiel
xx-small winzig 9px
x-small sehr klein 10px
small klein 13px
medium mittel (voreingestellte Schriftgröße des Browser) 16px
large groß 19px

xx-large riesig 32px
Em

Em: Viele Entwickler verwenden em anstelle von Pixeln, um den Benutzern zu ermöglichen, den Text in der Browsereinstellung zu vergrößern oder zu verkleinern. 1em entspricht der aktuellen Schriftgröße. Die Standardeinstellung für die Textgröße in den Browsern beträgt 16px. Daher ist die Standardgröße von 1em 16px. Die Größe kann von Pixeln zu em mit dieser Formel berechnet werden: pixels /16= em


h1 { font-size: 2.5em; /* 40px/16=2.5em */ }


h2 { font-size: 1.875em; /* 30px/16=1.875em */ }


p { font-size: 0.875em; /* 14px/16=0.875em */ }
Prozent & Em

Prozent und Em: Eine Lösung, die in allen Browsern funktioniert, besteht darin, eine Standard-Schriftgröße in Prozent für das <body>-Element festzulegen


body { font-size: 100%; }


h1 { font-size: 2.5em; }


h2 { font-size: 1.875em; }


p { font-size: 0.875em; }

font-style

Die font-style-Eigenschaft in CSS wird hauptsächlich verwendet, um kursiven Text anzugeben. Diese Eigenschaft hat drei Werte:

Normal

Der Text wird normal dargestellt.

p { font-style: normal; }

Hello world!

Italic

Der Text wird in Kursivschrift dargestellt.


p { font-style: italic; }

Hello world!

Oblique

Der Text wird “geneigt” dargestellt (Oblique ist sehr ähnlich zu Italic, wird aber weniger unterstützt).

p { font-style: oblique; }

Hello world!

font-weight

Die font-weight-Eigenschaft in CSS bestimmt die Dicke oder den Grad der Fettschrift. Hier sind die Werte, die Sie verwenden können:

Normal

Definiert normale Zeichen. Dies ist der Standardwert

p { font-weight: normal; }

Hello world!

Bold

Definiert dicke Zeichen

p { font-weight: bold; }

Hello world!

Oblique

Der Text wird “geneigt” dargestellt (Oblique ist sehr ähnlich zu Italic, wird aber weniger unterstützt).

p { font-style: oblique; }

Hello world!

Zahlenwerte

Ein Zahlenwert zwischen 1 und 1000, beide Werte eingeschlossen. Höhere Zahlen stellen Gewichte dar, die fetter sind als (oder genauso fett wie) niedrigere Zahlen. Dies ermöglicht eine feine Kontrolle für variable Schriften. Bei nicht variablen Schriften wird, wenn das genau angegebene Gewicht nicht verfügbar ist, ein Ausweichgewichtsalgorithmus verwendet

font-weight 100 Thin
font-weight 200 Extra-light
font-weight 300 Light
font-weight 400 regular
font-weight 500 Medium
font-weight 600 Semi-Bold
font-weight 700 Bold
font-weight 800 Extra-Bold
font-weight 900 Black
Lighter

Ein relatives Schriftgewicht, das leichter ist als das des Elternelements.

p { font-weight: lighter; }

Hello world!

Bolder

Ein relatives Schriftgewicht, das Schwerer ist als das des Elternelements.

p { font-weight: bolder; }

Hello world!

text-transform

Die text-transform-Eigenschaft in CSS bestimmt, wie der Text eines Elements hinsichtlich der Groß- und Kleinschreibung dargestellt wird. Hier sind die Werte, die Sie verwenden können:

None / Default

Keine Groß- oder Kleinschreibung. Der Text wird so dargestellt, wie er ist. Dies ist der Standardwert

p { text-transform: none; }

Hello world!

Capitalize

Wandelt den ersten Buchstaben jedes Wortes in Großbuchstaben um. Andere Zeichen bleiben unverändert (sie behalten ihre ursprüngliche Schreibweise, wie sie im Text des Elements geschrieben sind)

p { text-transform: capitalize; }

hello world!

hello world!

Uppercase

Wandelt alle Zeichen in Größbuchtaben um.

p { text-transform: uppercase; }

Hello world!

Lowercase

Wandelt alle Zeichen in Kleinbuchstaben um.

p { text-transform: lowercase; }

Hello world!

text-decoration

Die text-decoration-Eigenschaft in CSS bestimmt das Aussehen von dekorativen Linien auf Text. Es ist eine Kurzschreibweise für text-decoration-line, text-decoration-color, text-decoration-style und die neuere Eigenschaft text-decoration-thickness. Hier sind die Werte, die Sie verwenden können

text-decoration-line

Diese Eigenschaft wird verwendet, um eine Dekorationslinie zum Text hinzuzufügen. Sie können mehr als einen Wert kombinieren, wie overline und underline, um Linien sowohl über als auch unter einem Text anzuzeigen

p { text-decoration-line: overline; }

Hello world!

p { text-decoration-line: underline; }

Hello world!

text-decoration-color

Diese Eigenschaft wird verwendet, um die Farbe der Dekorationslinie festzulegen

p { text-decoration-line: overline; text-decoration-color: red;}

Hello world!

text-decoration-style

Diese Eigenschaft wird verwendet, um den Stil der Dekorationslinie festzulegen.

p { text-decoration-line: underline; text-decoration-style: wavy; }

Hello world!

text-decoration-thickness

Diese Eigenschaft wird verwendet, um die Dicke der Dekorationslinie festzulegen

p { text-decoration-line: underline; text-decoration-thickness: 25%; }

Hello world!

Shorthand Property

Die text-decoration Eigenschaft ist eine Kurzschreibweise für:

  • text-dexoration-line (erforderlich)
  • text-decoration-color (optional)
  • text-decoration-style (optional)
  • text-decoration-thickness (optional)
  • h1 { text-decoration: underline ; }
  • h2 { text-decoration: underline red ; }
  • h3 { text-decoration: underline red double; }
  • p { text-decoration: underline red double 5px; }

Hello world!

text-shadow

Die text-shadow- Eigenschaft in CSS fügt Textschatten hinzu. Jeder Schatten wird durch eine Kombination von X- und Y-Verschiebungen vom Element, Unschärferadius und Farbe beschrieben.
Hier ist die Syntax:

text-shadow

In diesem Beispiel:

  • 1px 1px sind die Horizontalen und vertikalen Verschiebungen des Schattens
  • 2px ist der Unschärferadius
  • black ist die Farbe des Schattens

/* offset-x | offset-y | blur-radius | color */

p {text-shadow: 1px 1px 2px black ; }

Hello world!

text-align

Die text-align- Eigenschaft in CSS wird verwendet, um die horizontale Ausrichtung des Textes innerhalb eines Elements zu steuern. Hier sind die Werte, die Sie verwenden können:

text-align
  • Left: Der Inhalt wird am linken Rand des Blocks ausgerichtet. Dies ist der Standardwert, wenn die Textausrichtung von links nach rechts ist
  • Right: Der Inhalt wird am rechten Rand des Blocks ausgerichtet. Dies ist der Standardwert, wenn die Textausrichtung von rechts nach links ist
  • Center: Der Inhalt wird in der Mitte des Blocks zentriert
  • Justify: Der Inhalt wird so ausgerichtet, dass die linke und rechte Kante des Blocks ausgefüllt sind. Jede Zeile hat die gleiche Breite, und die linken und rechten Ränder sind gerade (wie in Zeitschriften und Zeitungen)
  • Start: Das gleiche wie left, wenn die Richtung von links nach rechts ist, und right, wenn die Richtung von rechts nach links ist
  • End Das gleiche wie right, wenn die Richtung von links nach rechts ist, und left, wenn die Richtung von rechts nach links ist
p { text-align: left; } Left
p { text-align: right; } right
p { text-align: center; } Center
p { text-align: justify; } Justify
p { text-align: start; } Start
p { text-align: end; } End
