CSS Position-Eigenschaft

Einleitung

Die position-Eigenschaft in CSS wird verwendet, um die Positionierung eines Elements auf einer Webseite zu steuern. Sie definiert, wie ein Element relativ zu seinem normalen Fluss im Dokument positioniert werden soll. Zum Steuern der nicht statischen Elmente nutzt man die Eigenschaften: top, bottom, left und right um damit den Abstand anzugeben wie das Element in Abhängigkeit platziert werden soll.

static

Static ist der Standardwert der position Eigenschaft. Das Element folgt hierbei dem normalen Fluss des Dokuments.

static

relative

Das Element wird relativ zu seiner normalen Position verschoben. Andere Elemente im normalen Fluss werden nicht beeinflusst. In dieseim Beispiel wird das Element 100px nach rechts verschoben.

relative

absolute

Das Element wird relativ zum nächstgelegenen positionierten (nicht statischen) Elternelement positioniert. Wenn kein positioniertes Elternelement vorhanden ist, wird es relativ zum HTML-Element positioniert. Das Beispiel ist eine absolutes div in einem relativen div.

absolute

fixed

Das Element wird relativ zum Browserfenster positioniert und bleibt immer an derselben Stelle, auch wenn die Seite gescrollt wird. Das Beispiel hierfür ist am unteren rechten Bildschirmrand zu finden.

fixed

sticky

Ein Hybrid zwischen relative und fixed. Das Element wird relativ zum Viewport positioniert, bis es einen bestimmten Scrollpunkt erreicht, dann wird es fixiert. Wenn man den folgenden Text runterscrollt wird das sticky Element oben festgehalten und bleibt dort.

sticky
